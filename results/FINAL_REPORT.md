## §7 Provenance(來源檔 + 計算方式)

- **效能(§2/§5)**:系統 `results/<contract>/run{1..3}/system/eval/evaluation_results_system.csv`;baseline `.../baselines/<method>_baseline/eval/evaluation_results.csv`。
- **macro / micro / combined(§1)**:對每個 run i,把 Online124-run_i 與 Online39-run_i 配成一組(共 3 組)。**micro** = 兩份 TP/FP/FN 合計後算 P=TP/(TP+FP)、R=TP/(TP+FN)、F1;**macro** = 兩份各自 P/R/F1 再取平均。表中 TP/FP/FN/成本/時間皆為兩份合計。最後對 3 組取 mean±std[min–max]。
- **Funnel(§3/§4/§6)**:`results/<contract>/run{1..3}/system/analysis/funnel.json`。6a 輸入 = n_reg_norms × n_contract_norms,通過 = 交集非空配對數;6b 輸入 = 6a 通過,通過 = cosine≥thr survivors;6c 輸入 = 6b 通過,通過 = Compliant,篩除 = Violation+Gap。§3 為兩份逐 run 合計後取 mean[min–max]。
- **成本**:`.../system/cost/main_usage.json`、`.../baselines/<m>_baseline/cost/<m>_usage.json` 的 token 用量 × `config.yaml` pricing。**時間**:`.../system/time/step_times.json`、`.../baselines/<m>_baseline/time/time.json`。
- **參數 / KG db**:各 run 的 `results/<contract>/run<N>/run_meta.json`。
